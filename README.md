# Bernhoeft
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://github.com/felipepaganin/bernhoeft/blob/master/license)

Esse é um projeto criado em .NET 7, utilizando sql server e Entity framework.

O projeto foi criado atraves de um desafio técnico proposto pela empresa.

## Funcionalidades.
As principais funcionalidades são:

![image](https://user-images.githubusercontent.com/64207566/229180872-cd0d1a61-a40b-4dca-910a-45dfdca5d71a.png)

1. Cadastrar categoria
2. Alterar categoria
3. Listar categorias (Deve haver a possibilidade de filtrar por nome da categoria e situação
da categoria)
4. Cadastrar produto
5. Alterar produto
6. Listar produtos (Deve haver a possibilidade de filtrar por categoria do produto, descrição
do produto e situação do produto)


## Como executar o projeto.
Para executar o projeto, você seguirá as senguintes etapas:

1. Clone esse repositorio em sua maquina local, usando o comando git clone https://github.com/felipepaganin/bernhoeft.git.
2. Abra o projeto em sua IDE de preferencia.
3. Defina o projeto WebApi como projeto de inicialização.
4. Abra o arquivo appsettings.development.
5. Edite as configuração com o banco de dados de acordo com sua instância local
6. Compile e execute o projeto.
7. Caso não abra o swagger automaticamente, compile o projeto e entre no link: https://localhost:7154/swagger/index.html.
